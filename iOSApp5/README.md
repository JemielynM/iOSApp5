# iOSApp5 – Media Moments

Media Moments is a SwiftUI application that allows users to select, display, play, and manage photos and videos.

## Features

- Select photos from the photo library
- Select and play videos
- Add titles and captions
- Display media in a gallery
- View media details
- Swipe to delete
- Confirmation alerts
- Tab-based navigation
- NavigationStack
- Sheets and forms
- Empty-state interface

## Technologies

- Swift
- SwiftUI
- PhotosUI
- AVKit
